🎓 Student Management System (Tkinter)

A simple Student Management System built using Python Tkinter.
This application allows users to Add Student, View Student, and Delete All Data.


---

📌 FEATURES

✅ Add Student Details
✅ View Saved Students
✅ Delete All Student Records
✅ Simple GUI using Tkinter
✅ File-based Data Storage


🖥️ APPLICATION PREVIEW

Main Options:

1. Add Student


2. View Student


3. Delete All Data




---

📂 PROJECT STRUCTURE

StudentManagementSystem/
│
├── student.py
├── SMS.RECORD
└── README.md


---

⚙️ REQUIREMENTS

Make sure you have Python installed:

Python 3.x
Tkinter (Pre-installed with Python)

Check Python Version:

python --version


---

▶️ How To Run

Step 1: Download or Clone Repository

git clone https://github.com/yourusername/student-management-system.git

Step 2: Go to Project Folder

cd student-management-system

Step 3: Run Program

python student.py


---

🧾 DATA STORAGE

Student data is saved inside:

SMS.RECORD

Example:

John, Smith, Mary, 9876543210
Rahul, Kumar, Sita, 9876543211


---

📌 FUNCTIONS USED

Add Student

Allows user to enter:

Name

Father Name

Mother Name

Mobile Number



---

View Student

Displays all saved student records.


---

Delete All Data

Deletes entire student record file.
